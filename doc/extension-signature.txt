# Extension: Signature (single|multiple)

place
: Where was the document produced.

date
: When was the document produced.

sign
: Provide data about the Person signing. Assigning multiple values
  may be supported, depending on the purpose of the template.
  Also check if it refers to *Signature (single)* or
  *Signature (multiple)*.
: If you want to include lines, you can use the ``\newline`` command
  from latex. This can usually be abbreviated as ``\n`` in these cases.
